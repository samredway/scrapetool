Scrape Tool

Enter in the form below the URL of the page you want to scrape and a prompt for the AI to use when scraping

Advanced users can specify a JSON string to define the response structure. For example the default response (shown below) is an array of strings.

This tool uses github.com/samredway/scrapeai under the hood. For more information on the response structure, see the docs here: ScrapeAi 

{
    "type": "object",
    "properties": {
        "data": {
            "type": "array",
            "items": {"type": "string"}
        }
    },
    "additionalProperties": false,
    "required": ["data"]
}
